# Reports

One-shot quantitative or qualitative snapshots. Each report captures
measurements + findings + recommendations at a point in time. See
[`../operations/doc_conventions.md`](../operations/doc_conventions.md#type-report)
for the canonical structure.

Reports are time-bound by nature; filename convention is
`YYYY-MM-DD_topic-slug.md`.
