# What changed for the office (G2.3)

- **Subcontractors can now fix their own crew typos** (edit name/trade) and retire crew *they
  created* — but only when nobody else has logged time against the person and the office hasn't
  moved them to another job. Anything else still routes to you.
- **Time entries are now correctable — nothing is ever deleted.** A correction shows as the current
  row with a **"corrected" pill** (the history is preserved underneath); a **void** shows
  struck-through with the stated reason and counts as 0 hours. Labor totals always use the latest
  version of each entry.
- If a weekly number changes after the fact, look for the corrected/voided marks — that's an
  auditable field correction, not data loss.
