# Enablement docs (§6 / A8)

Operator- and end-user-facing **enablement guides** — plain-language manuals for the people who
*use* an ITS capability (the office PM, field crews, stakeholders), as opposed to the
code-reader `§42` docstrings or the successor-operator `§43` runbooks in
[`../runbooks/`](../runbooks/).

These are the in-repo source of truth for the **§6 / A8 documentation program** (every ITS
function gets a guide/manual; the polished distributable PDF versions land before each cutover).
Each guide uses `type: operations` frontmatter (the conforming type for a how-to doc — the
convention has no separate `enablement` type; see
[`../operations/doc_conventions.md`](../operations/doc_conventions.md)).
